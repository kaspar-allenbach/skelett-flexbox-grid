Gugus Grid

This is a minimal grid framework. The Flexbox Grid is veryvery similar to the bootstrap grid but without the overhead. It is supposed to be a bare-bones grid which you can adapt to your needs without framework bloat. Just a flexbox & css grid and thats it. (Gugus: Peek-a-Boo in Swiss German)

Flexbox Grid

Define the amount of Columns via $baseGrid. This will generate all classes you need.
Example: $baseGrid: 15; will generate a grid with 15 Columns and all related cllasses for you to work with.
Media Queries: All media query related classes use gugus-media-queries. See the docs for Media Breakpoints.

Wrappers

There ar also wrapper classes to contain the grids.

There are three wrapper types: .innerWrapper, .outerWrapper, .fullWrapper

All Wrappers lock at the size: $baseWrapSize

All wrappers are the same width on mobile with a padding size of $gridPadding
