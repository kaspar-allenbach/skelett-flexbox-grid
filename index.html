Gugus Grid

This is a minimal grid framework. The Flexbox Grid is veryvery similar to the bootstrap grid but without the overhead. It is supposed to be a bare-bones grid which you can adapt to your needs without framework bloat. Just a flexbox & css grid and thats it. (Gugus: Peek-a-Boo in Swiss German)

Flexbox Grid

Define the amount of Columns via $baseGrid. This will generate all classes you need.
Example: $baseGrid: 15; will generate a grid with 15 Columns and all related cllasses for you to work with.
Media Queries: All media query related classes use gugus-media-queries. See the docs for Media Breakpoints.

Wrappers

There ar also wrapper classes to contain the grids.

There are three wrapper types: .innerWrapper, .outerWrapper, .fullWrapper

All Wrappers lock at the size: $baseWrapSize

All wrappers are the same width on mobile with a padding size of $gridGap

Flexbox Container Classes

.o-flex
Initiates the Flexbox Library
.o-flex-mediaQuery
Initiates the Flexbox Library based on screen size (.sm__up, .md__up, .lg__down Media Query Library )
.o-row-col-#
Add the grid with to the parent element if you don't want to define it on the child

Flexbox Children Classes

.o-col
Standard class. Makes equal width elements
.o-col-#
Creates element width of # (Default to 12 grid)
.o-col-mediaQuery-#
Apply media query based element width's

Flexbox directional Classes

.u-direction-#
Applies flex-direction parameters. Use css property syntax for # (row, row-reverse …)
.u-direction-mediaQuery-#
Applies Media Query based directional directives

Flexbox Spacing Classes

Spacing classes start with .u-x- and .u-y- signaling the X and Y axis.

.u-x-#
Applies justify-content parameters. Use css property syntax for # (space-around, space-between…). These classes only work when .u-col-basis-auto is applied to the parent (removing flex-basis: 1)
.u-x-mediaQuery-#
Applies Media Query based directional directives
.u-y-#
Applies align-items to the child elements. Use css property syntax instead of
.u-y-mediaQuery-#
Blablabla Responsive blablabla

Flexbox Order Classes

.u-col-order-#
Reorder elements visually
.u-col-order-mediaQuery-#
Reorder elements visually based on media queries

Special Utilities

.u-col-basis-auto
Sets back flex-basis meaning the elements will default to inner content width. Nexessery for Y-axis spacing.

Shrink & Grow Utilities

.u-shrink-true
Allowes element to shrink
.u-shrink-false
Doesn't allow element to shrink
.u-grow-true
Allowes element to grow
.u-grow-false
Doesn't allow element to grow
.u-grow-mediaQuery-#
Also possible to use with media queries

Gap Utilities

.u-gap
Ads a gap to all children of .o-flex with padding to the child and negative margin on the parent (same hack as bootstrap). gap size is $gridGap
.u-gap-mediaQuery
Mediaquery based grid gap
